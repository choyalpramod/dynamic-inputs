# React + TypeScript + Vite

## Steps

Step 1: Open terminal[tab 1]

```
npm -g install json-server
```

Step 2:

```
json-server --watch task/input.json --port 3001
```

Step 3: Open terminal[tab 2]

```
npm install
```

or

```
pnpm install
```

Step 4:

```
npm run dev
```

or

```
pnpm dev
```

Step 5: Go to [localhost](http://localhost:3000/)
